📋 10 BƯỚC NHANH:

1️⃣  Tạo project Google Cloud
    → https://console.cloud.google.com/

2️⃣  Enable Google Drive API
    → APIs & Services > Library > Search "Drive"

3️⃣  Tạo Service Account
    → APIs & Services > Credentials > Create > Service Account

4️⃣  Download JSON key
    → Click vào Service Account > Keys > Add Key > JSON
    → Đổi tên file thành: google-credentials.json

5️⃣  Copy Service Account Email
    → Từ file JSON, tìm "client_email"

6️⃣  Share Drive folder
    → https://drive.google.com/drive/folders/1OBWCc80JfExTWyUxx3W2I6ltY8CZILa0
    → Share với Service Account Email
    → Quyền: Editor

7️⃣  Upload google-credentials.json
    → cPanel File Manager
    → /public_html/api/google-credentials.json
    → Permissions: 600

8️⃣  Edit config.php
    → GOOGLE_DRIVE_ENABLED = true
    → GOOGLE_DRIVE_FOLDER_ID = 1OBWCc80JfExTWyUxx3W2I6ltY8CZILa0

9️⃣  Cài Google API Client
    → Via SSH: composer require google/apiclient
    → Hoặc upload vendor/ manually
    → Hoặc yêu cầu hosting support

🔟 Test
    → Upload test_google_drive.php
    → Truy cập: https://qlvb.phongkhcn.vn/api/test_google_drive.php
