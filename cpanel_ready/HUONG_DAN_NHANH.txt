BƯỚC 1: IMPORT DATABASE
========================================

2. Vào "MySQL® Databases"
3. Tạo database mới: qlvb_db
4. Tạo user: qlvb_user (+ password)
5. Thêm user vào database với ALL PRIVILEGES
6. Vào "phpMyAdmin"
7. Chọn database qlvb_db
8. Tab "Import"
9. Chọn file: database.sql
10. Click "Go"

✅ Database sẵn sàng!

========================================
BƯỚC 3: CẤU HÌNH DATABASE
========================================

1. File Manager > /public_html/api/config.php
2. Click phải > Edit
3. Tìm dòng:

   define('DB_NAME', 'qlvb_db');
   define('DB_USER', 'root');
   define('DB_PASS', '');

4. Đổi thành (CHÚ Ý PREFIX):

   define('DB_NAME', 'username_qlvb_db');  // Thêm username_ ở đầu
   define('DB_USER', 'username_qlvb_user'); // Thêm username_ ở đầu
   define('DB_PASS', 'password_cua_ban');   // Password bạn đã tạo

   VÍ DỤ: Nếu username cPanel là "mysite":
   define('DB_NAME', 'mysite_qlvb_db');
   define('DB_USER', 'mysite_qlvb_user');
   define('DB_PASS', 'MyP@ssw0rd123');

5. Tìm dòng JWT_SECRET và đổi thành string ngẫu nhiên:

   define('JWT_SECRET', 'abc123xyz789-doi-thanh-chuoi-ngau-nhien');

6. Click "Save Changes"

========================================
BƯỚC 4: SET PERMISSIONS
========================================

1. Chọn tất cả folders trong /public_html/
2. Click "Permissions"
3. Set: 755 (rwxr-xr-x)
4. Check "Recurse into subdirectories"
5. Click "Change Permissions"

6. Chọn tất cả FILES (.php, .html, .htaccess)
7. Set: 644 (rw-r--r--)

========================================
TROUBLESHOOTING - NẾU CÓ LỖI
========================================

❌ LỖI: 500 Internal Server Error
✅ GIẢI QUYẾT:
   - Check PHP version >= 7.4
   - Check file .htaccess có đúng
   - Xem error log: cPanel > Errors > Error Log

❌ LỖI: Database connection failed
✅ GIẢI QUYẾT:
   - Kiểm tra lại DB_NAME, DB_USER, DB_PASS
   - Nhớ thêm PREFIX username
   - Verify user có ALL PRIVILEGES

❌ LỖI: 404 Not Found
✅ GIẢI QUYẾT:
   - Check .htaccess đã upload
   - Verify mod_rewrite enabled

❌ LỖI: Cannot read property...
✅ GIẢI QUYẾT:
   - Clear browser cache (Ctrl + Shift + Delete)
   - Hard refresh (Ctrl + F5)

🔒 3. Enable HTTPS redirect:
   - Thêm vào .htaccess đầu file:
   
   RewriteCond %{HTTPS} off
   RewriteRule ^(.*)$ https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301]
